📌**campus placement preparation platform** built specifically for Indian engineering students. It helps students:

- **Know where they stand** — a weighted scoring algorithm predicts placement probability across 8 profile parameters
- **Identify weak spots** — instant, specific insights on CGPA, skills, internships, and projects
- **Get coached** — a keyword-aware AI coach answers placement questions with actionable advice
- **Practice interviews** — a mock interview engine simulates technical, HR, and system design rounds
- **Build a roadmap** — generates a personalized week-by-week study plan based on your target role and timeline
- **Fix their resume** — ATS analyzer scores resumes and rewrites weak bullet points


AI-powered campus placement predictor with a real Python Flask backend.

---


## Tech Stack

| Layer | Technology |
|---|---|
| Backend | Python 3.11+, Flask 3.0 |
| HTTP Client | Python `requests` library |
| Storage | JSON file (upgradeable to PostgreSQL/MongoDB) |
| Frontend | Vanilla JS, HTML5, CSS3 |
| Fonts | Google Fonts (Syne + DM Sans) |

🚀 Live Demo
Open http://localhost:5000 after running the server — the frontend is served directly by Flask.
